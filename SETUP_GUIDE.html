① Neon Database

done ~3 min

Without this, nothing gets saved and the Conflict Checker can't compare against existing pages.

Open Neon ↗
  1. Go to console.neon.tech/signup and sign in with Google (marketing@edstellar.com). Free tier is fine.
  2. Click "Create a project".
    • Name: conflict-checker
    • Postgres: default (17)
    • Region: closest to you (Asia Pacific = Singapore/Mumbai)
  3. On the project dashboard find "Connection string". Toggle Pooled connection on, then click copy.
  4. Open .env and paste it: 
    DATABASE_URL=postgresql://neondb_owner:...@ep-...-pooler.region.aws.neon.tech/neondb?sslmode=require
  5. Save the file. Tell me "Neon done" — I run npm run db:setup and start ingesting.

② Google Search Console OAuth

enables Search Console tab ~5 min

Lets you pull GSC performance data (clicks, impressions, CTR, position) for any window from 24h to 12 months. Skip if you only want conflict checking.

Open Google Cloud Console ↗ Enable Search Console API ↗
  1. Open console.cloud.google.com. Sign in with the Google account that owns Search Console for edstellar.com.
  2. Top bar → project dropdown → "New Project". Name conflict-checker, create, then select it.
  3. Enable the API: open this link → click Enable.
  4. Configure the OAuth consent screen: credentials/consent
    • User type: External → Create
    • App name: Conflict Checker
    • Support email / Developer contact: marketing@edstellar.com
    • Skip Scopes. On Test Users, add marketing@edstellar.com.
    • Save and continue → Back to Dashboard.
  5. Create the OAuth client: credentials → + Create Credentials → OAuth client ID
    • Application type: Web application
    • Name: Conflict Checker Local
    • Authorized redirect URI: 
      http://localhost:3000/api/gsc/callback
    • Create.
  6. Copy the Client ID and Client Secret from the popup, paste into .env: 
    GOOGLE_CLIENT_ID=xxxxxxxxxx.apps.googleusercontent.com
GOOGLE_CLIENT_SECRET=GOCSPX-xxxxxxxx
  7. Save. The Connect Google button on the Search Console tab will now work.

③ Serper

enables Competitors tab ~2 min

Runs Google searches for a topic to find competitor pages. Free tier = 2,500 searches.

Open Serper ↗
  1. Go to serper.dev → Sign up with Google.
  2. On the Dashboard, copy the API Key shown at the top.
  3. Paste into .env: 
    SERPER_API_KEY=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
  4. Save.

④ Claude API

optional ~2 min

You already have Groq (fast). Claude writes higher-quality summaries / conflict rationales — at the cost of speed.

Open Anthropic Console ↗
  1. Go to console.anthropic.com, sign up.
  2. Settings → API Keys → Create Key. Copy (starts with sk-ant-).
  3. Paste into .env: 
    ANTHROPIC_API_KEY=sk-ant-...
  4. To switch from Groq to Claude: 
    AI_CHAT_PROVIDER=claude
